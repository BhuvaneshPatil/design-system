# Searchbar

SearchBars are used to search and navigate through specific content. 

**Best practices**

- SearchBar should not be used at a global application level.
- SearchBar can come with a PopOver component to access the list of suggested options.
- The search icon nor the cancel one should be changed.
- SearchBar should always have a "Search" placeholder.
- SearchBar should be placed above the content to search into.
- SearchBar do not have a label.
- If adding a SearchBar, make sure to add a "No results found" state.
