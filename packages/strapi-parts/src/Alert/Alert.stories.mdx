### Alert variants

Use the variant prop to change the visual style of the Button.
You can set the value to `default`, `success`, or `danger`.

<Canvas withSource="open">
  <Story
    name="variants"
    parameters={{

          code: '<Alert title="Title" variant="success">This is the default variant.</Alert>',
        },
      },
    }}
  >
    <Box style={{ width: 700 }}>
      <Stack size={1}>
        <Alert title="Title">This is the default variant.</Alert>

          This is the danger variant.
        </Alert>
      </Stack>
    </Box>
  </Story>
