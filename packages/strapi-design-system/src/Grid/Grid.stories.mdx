# Grid

This is the doc of the `Grid` component

## Grid

The component allows to create a Grid on the screen using the CSS grid feature. This component extends the `<Box />` one and can receive the same props.

<Canvas>
  <Story name="base">
    <Grid gap={5}>
      <Box col={1} background="primary100">
        First
      </Box>
      <Box col={1} background="primary100">
        Second
      </Box>
      <Box col={1} background="primary100">
        Third
      </Box>
      <Box col={1} background="primary100" color="primary700">
        Last
      </Box>
    </Grid>
  </Story>
</Canvas>
