### Button sizes

Use the size prop to change the size of the button. You can set the value to `S` or `L`.

<Canvas withSource="open">
  <Story
    name="sizes"
    parameters={{
      docs: {
        source: {
          code: `
<Button size="S">Small</Button>
<Button size="L">Large</Button>
`,
        },
      },
    }}
  >
    <Row>
      <Box paddingRight={1}>
        <Button size="S">Small</Button>
      </Box>
      <Button size="L">Large</Button>
    </Row>
  </Story>
</Canvas>

### Button variants

Use the variant prop to change the visual style of the Button.
You can set the value to `default`, `secondary`, `tertiary`, `success`, `danger`, `success-light`, or `danger-light`.

<Canvas withSource="open">
  <Story
    name="variants"
    parameters={{
      docs: {
        source: {
          code: `
<Button variant='default'>Default</Button>
<Button variant='secondary'>Secondary</Button>
<Button variant='tertiary'>Tertiary</Button>
...
  `,
        },
      },
    }}
  >
    <Row>
      {['default', 'secondary', 'tertiary', 'success', 'danger', 'success-light', 'danger-light'].map((variant) => (
        <Box key={variant} paddingRight={2}>
          <Button onClick={(e) => console.log(e)} variant={variant}>
            {variant}
          </Button>
        </Box>
      ))}
    </Row>
  </Story>
</Canvas>

### Button with icon

You can add start and end icons to the Button component using the `startIcon` and `endIcon` props respectively.

<Canvas withSource="open">
  <Story
    name="icons"
    parameters={{
      docs: {
        source: {
          code: `
<Button startIcon={<HelpIcon />}>Information</Button>
<Button variant='secondary' endIcon={<ContentIcon />}>Write content</Button>
`,
        },
      },
    }}
  >
    <Row>
      <Box paddingRight={1}>
        <Button startIcon={<HelpIcon />}>Information</Button>
      </Box>
      <Button variant="secondary" endIcon={<ContentIcon />}>
        Write content
      </Button>
    </Row>
  </Story>
</Canvas>

## Accessibility

- Button has role of button.
- When Button has focus, `Space` or `Enter` activates it.
